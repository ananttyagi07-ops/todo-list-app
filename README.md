# todo-list-app
A simple To-Do List web application developed using HTML, CSS, and JavaScript. This project allows users to add and manage daily tasks through a clean and interactive interface.  

## Features
- Add tasks dynamically
- Interactive user interface
- Input validation
- Simple and responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Concepts Learned
- DOM Manipulation
- Functions
- Event Handling
- User Input Handling
